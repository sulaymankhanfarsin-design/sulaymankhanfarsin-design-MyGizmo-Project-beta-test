How to Use the API

Use the X-API-Key header to authenticate all requests. Select a tool below to see the code example:

1. Slug Generator (POST):

curl -X POST "{{ url_for('api_slugify', _external=True) }}" \
-H "Content-Type: application/json" \
-H "X-API-Key: YOUR_API_KEY_HERE" \
-d '{
    "text": "This is my Example Text!",
    "separator": "_",
    "remove_numbers": true
}'

2. QR Code Generator (POST):

curl -X POST "{{ url_for('api_qr_generator', _external=True) }}" \
-H "X-API-Key: YOUR_API_KEY_HERE" \
-F "text=https://mygizmo.com" \
-F "fill_color=#0000FF" \
-F "back_color=#FFFFFF" \
-F "logo_image=@/path/to/your/logo.png" \
--output my_custom_qr.png

3. AI Background Remover (POST):

curl -X POST "{{ url_for('api_bg_remover', _external=True) }}" \
-H "X-API-Key: YOUR_API_KEY_HERE" \
-F "image=@/path/to/your/image.jpg" \
--output removed_bg.png

4. List Randomizer (POST):

curl -X POST "{{ url_for('api_list_randomizer', _external=True) }}" \
-H "Content-Type: application/json" \
-H "X-API-Key: YOUR_API_KEY_HERE" \
-d '{
    "items": ["Apple", "Banana", "Orange", "Grape"],
    "num_selections": 2
}'

5. JPG to PDF Converter (POST):

curl -X POST "{{ url_for('api_jpg_to_pdf', _external=True) }}" \
-H "X-API-Key: YOUR_API_KEY_HERE" \
-F "images=@/path/to/image1.jpg" \
-F "images=@/path/to/image2.png" \
--output converted.pdf

6. PDF to JPG Converter (POST):

curl -X POST "{{ url_for('api_pdf_to_jpg', _external=True) }}" \
-H "X-API-Key: YOUR_API_KEY_HERE" \
-F "file=@/path/to/document.pdf" \
--output converted_images.zip

7. Image Studio (POST):

curl -X POST "{{ url_for('api_image_studio', _external=True) }}" \
-H "X-API-Key: YOUR_API_KEY_HERE" \
-F "images=@/path/to/image1.jpg" \
-F "width=800" \
-F "rotate=90" \
-F "flip=horizontal" \
-F "img_filter=grayscale" \
-F "watermark_text=© MyGizmo" \
-F "output_format=PNG" \
--output processed_images.zip
